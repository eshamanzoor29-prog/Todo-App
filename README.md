# ✅ Todo App

A modern and fully responsive Todo App built with HTML, CSS, and JavaScript. The application allows users to efficiently manage daily tasks through an intuitive interface with dynamic task management and persistent data storage. It demonstrates strong front-end development skills, interactive functionality, and clean code organization.

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All, Active, Completed)
- Clear completed tasks
- Drag and drop to reorder tasks
- Data persistence using LocalStorage
- Responsive design for all devices
- Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
